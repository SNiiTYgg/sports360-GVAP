# 🏆 campus360sports

A modern, responsive web platform for showcasing **college sports activities** in one place.

---

## 📌 About the Project

**campus360sports** is a web-based platform developed for **GVAP College** to bring together sports activities of all houses on a single digital platform. It allows houses to showcase images and videos from sports events, engage students through interactive polls, and display scorecards in a clean and modern interface.

The project is designed to be **simple**, **mobile-friendly**, and **easy to use**, making it ideal for campus-wide sports engagement.

---

## 🏠 Houses Included

The platform supports the following houses:

- 🔵 **AAKASH**
- 🔴 **AGNI**
- 🔵 **JAL**
- 🟢 **PRITHVI**
- ⚪ **VAYU**

Each house has its own dedicated section with media and details.

## ✨ Key Features

📸 **Media Showcase**  
- Display images and videos from sports events  
- Instagram-style media grid  
- Fullscreen modal view with descriptions  

📊 **Interactive Polls**  
- Polls for student participation  
- One vote per device/browser  
- Clean poll cards inspired by Instagram  

🏆 **Scorecards**  
- Display match or event scorecards  
- Easy-to-read format  

🌗 **Light & Dark Mode**  
- Theme toggle for better accessibility  

📱 **Responsive Design**  
- Works smoothly on desktop and mobile devices

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:8080`.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run build:dev` - Build for development
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build
